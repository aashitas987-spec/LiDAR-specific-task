# LiDAR-specific-task
The DEM was obtained from OpenTopography and loaded using Rasterio. A hillshade was then generated by taking terrain slope and aspect and simulating illumination from a light source with azimuth 315° and altitude 45°. 
There are some dark and light patches showing that many slopes face in different directions (receive different amounts of light) so the ground must be having ridges, shadow valleys and irregular terrain. 
Image is also highly textured which may reflect vegetation or surface objects because the image input is a DSM which is why the features are all above the ground surface features, they can create clustered patterns.
